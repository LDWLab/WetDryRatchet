Chemical Oligomer Reaction Network Simulator
This project simulates chemical reaction networks of oligomer formation under non-equilibrium environmental conditions. The model includes condensation, hydrolysis, and exchange reactions among oligomers of arbitrary length, with temperature- and water-dependent kinetics. It can simulate dry–wet cycling, temperature perturbations, molecular feeding, and general reaction networks. The framework also contains optional modules for catalytic reactions that are not used in the present work.
File Structure
Core File
oligomer_22_ratchet.py
Main simulation program containing:
Generation of all oligomer sequences up to a specified maximum length.
Construction of condensation, hydrolysis, and exchange reaction networks.
Temperature-dependent reaction kinetics using transition state theory.
Thermodynamic equilibrium constants.
Dry–wet cycling through variable water concentration.
Time-dependent temperature protocols.
Molecular feeding.
Numerical integration of the complete reaction network.
Saving and plotting of simulation results.

Usage
1. Configure the simulation
All simulation parameters are located in the User parameters section near the bottom of the script.
The most commonly modified parameters include:
System size
M — number of monomer types
L_max — maximum oligomer length
init_length — length of initially populated species
Initial conditions
C0_total
monomer_C0
Simulation time
t_span
time_scale
n_points
Thermodynamic parameters
deltaG_act_kcal
deltaG_std_kcal
deltaG_exch_act_kcal
Water protocol
water_mode
water_params
Available water modes include:
constant
square
custom
The square-wave protocol allows simulation of repeated dry–wet cycles.
Temperature protocol
temp_mode
temp_params
Available temperature modes include:
constant
piecewise
cycle_with_perturbation

Exchange reactions
Exchange chemistry can be enabled or disabled using

enable_exchange = True

enable_exchange = False

Molecular feeding
Feeding can be enabled using

FEED_ENABLED = True

with
feed_times_model
feed_amounts
feeding_type
Note: The code also contains optional modules for several catalytic mechanisms (reaction-specific catalysis, species-mediated catalysis, and hydrolysis protection), although these are not used in the present study.
2. Run the simulation
Execute

python3 oligomer_22_ratchet.py

The program will
generate all sequences,
construct the reaction network,
integrate the kinetic equations,
print simulation statistics,
optionally display concentration plots,
save concentration trajectories.

To plot, execute
python3 plot_code.py
Plotting utility that reads timeseries_values.txt generated by the simulation and produces:
Time evolution of selected or all species.
Final concentration bar plots.
Publication-quality figures with the same color scheme used by the simulator.


Output
The simulation reports
total number of species
number of condensation/hydrolysis reactions
number of exchange reactions
If enabled, the program writes

timeseries_values.txt

which contains
simulation parameters
physical time values
concentration of every species throughout the simulation
Concentrations may optionally be reported after correcting for changing solution volume during dry–wet cycling.
Model Description
The reaction network consists of three classes of reactions.
Condensation / Hydrolysis

A + B ⇌ AB + H₂O

Forward and reverse rate constants are calculated from transition state theory and thermodynamic equilibrium.
Exchange

AB + C ⇌ AC + B

Exchange reactions are generated automatically for all compatible oligomers.
Environmental Driving
Reaction kinetics may depend on
temperature
water concentration
water activity
The model supports repeated dry–wet cycling as well as arbitrary environmental protocols.
